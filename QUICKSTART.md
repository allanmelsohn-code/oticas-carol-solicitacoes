## 🎯 Casos de Uso Rápidos

### Criar uma Solicitação de Montagem
1. Menu → **Nova Solicitação**
2. Selecione a loja
3. Escolha "Montagem (Laboratório)"
4. Preencha OS, valor e justificativa
5. Clique em **Enviar para Aprovação**

### Aprovar uma Solicitação
1. Menu → **Aprovações**
2. Veja a lista de pendentes
3. Clique no ✓ verde para aprovar
4. Ou ✕ vermelho para reprovar (com observação)

### Gerar Relatório Mensal
1. Menu → **Extrato Mensal**
2. Selecione mês/ano e filtros
3. Clique em **PDF** para baixar relatório
