# PWA — Óticas Carol — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Tornar o app Óticas Carol instalável na tela inicial de celulares Android e iPhone, abrindo em modo standalone sem barra de endereço.

**Architecture:** Usamos `vite-plugin-pwa` para gerar Web App Manifest e Service Worker mínimo (NetworkOnly — sem cache offline). Os ícones são gerados a partir da imagem-fonte `Gemini_Generated_Image_i4cgy5i4cgy5i4cg.png` via script com `sharp`. Tags iOS são adicionadas diretamente ao `index.html`.

**Tech Stack:** Vite, vite-plugin-pwa, workbox (via vite-plugin-pwa), sharp (script one-off), pnpm

## Task 2: Gerar ícones

**Files:**
- Create: `scripts/generate-icons.mjs`
- Create: `public/icon-192x192.png` (gerado)
- Create: `public/icon-512x512.png` (gerado)
- Create: `public/apple-touch-icon.png` (gerado)

- [ ] **Step 1: Criar script de geração de ícones**

Criar `scripts/generate-icons.mjs` com o conteúdo:

```js
import sharp from 'sharp'

const source = 'Gemini_Generated_Image_i4cgy5i4cgy5i4cg.png'

await sharp(source).resize(192, 192).toFile('public/icon-192x192.png')
await sharp(source).resize(512, 512).toFile('public/icon-512x512.png')
await sharp(source).resize(180, 180).toFile('public/apple-touch-icon.png')

console.log('Ícones gerados em public/')
```


Saída esperada:

## Task 3: Configurar vite-plugin-pwa

**Files:**
- Modify: `vite.config.ts`

- [ ] **Step 1: Atualizar vite.config.ts**

Substituir o conteúdo de `vite.config.ts` por:

```ts
import { defineConfig } from 'vite'
import path from 'path'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'
import { VitePWA } from 'vite-plugin-pwa'

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(),
    VitePWA({
      registerType: 'autoUpdate',
      includeAssets: ['favicon.svg', 'apple-touch-icon.png'],
      manifest: {
        name: 'Óticas Carol',
        short_name: 'Óticas Carol',
        description: 'Gerenciamento de solicitações da Óticas Carol',
        theme_color: '#111827',
        background_color: '#ffffff',
        display: 'standalone',
        start_url: '/',
        icons: [
          {
            src: 'icon-192x192.png',
            sizes: '192x192',
            type: 'image/png',
          },
          {
            src: 'icon-512x512.png',
            sizes: '512x512',
            type: 'image/png',
            purpose: 'any maskable',
          },
        ],
      },
      workbox: {
        runtimeCaching: [],
      },
    }),
  ],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src'),
    },
  },
  assetsInclude: ['**/*.svg', '**/*.csv'],
})
```

- [ ] **Step 2: Verificar que o TypeScript não tem erros**

```bash

```

Saída esperada: sem erros.

- [ ] **Step 3: Commit**


## Verificação manual no celular

Após deploy no Vercel:

**Android (Chrome):**
1. Abrir o app no Chrome
2. Menu (⋮) → "Adicionar à tela inicial" deve aparecer, ou o banner de instalação aparece automaticamente
3. Instalar e abrir — deve abrir sem barra de endereço

**iPhone (Safari):**
1. Abrir o app no Safari
2. Botão de compartilhar → "Adicionar à Tela de Início"
3. Instalar e abrir — deve abrir sem barra de endereço com o ícone correto
